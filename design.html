② 소요시간 계산프로젝트가 8일 안에 가능한지 확인해보기
③ 오차의 영향 확인우리가 계획한 조건들이, 측정 오차 때문에 서로 구별이 안 되는 건 아닐까?

만약 온도를 조작 변인으로 잡고, 20°C / 22°C / 24°C 세 조건에서 반응 속도를 비교하는 실험을 계획했다고 해봅시다. 그런데 우리가 쓰는 온도계가 ±3°C 오차를 가진 저렴한 온도계라면 어떨까요?

20°C라고 표시된 게 실제로는 17~23°C 사이 어딘가일 수 있고 24°C라고 표시된 것도 실제로는 21~27°C 사이 어딘가일 수 있습니다.

→ 이 경우 20°C 조건과 24°C 조건이 실제로는 겹칠 수도 있다는 뜻입니다. 세 조건을 나눠서 실험해도, "온도 차이 때문에 속도가 달라졌다"고 자신 있게 말할 수 없게 됩니다. 오차가 조건 간 차이를 삼켜버린 거예요.

더 헷갈리면 예시로 확인하기 ▾
판단 순서 — 내 계산식을 보고 물어보세요
  1. 나누기(÷)가 있나요? 없다 → 차이형 (예: ΔT = T₂ − T₁)
  2. 있다면, 나누는 분모가 조작 변인 자체인가요? 그렇다 → 역수형 (예: Δ = hc/λ)
  3. 아니라면, 서로 다른 두 측정값의 비율인가요? 그렇다 → 비율형 (예: ε = A/(c·l))
숫자로 확인하기
λ = 500 nm, 오차 ±10 nm라면
Δ = hc/λ → λ가 490~510 사이 어디든 될 수 있음
→ λ가 커지면 Δ는 작아진다(반비례) → 역수형
❌ 착각하기 쉬운 지점
"둘 다 나눗셈이니 같은 거 아닌가?" → λ는 조작 변인 자체가 분모라 그래프가 휘고(역수), c·l은 서로 독립적인 두 값의 몫이라 그래프가 직선(비율)입니다.
